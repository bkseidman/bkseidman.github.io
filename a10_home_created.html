Using an invented dataset on student sleep patterns

Assignment 10 — Misleading Visualizations

Target Grade: B

Overview

This project demonstrates how data visualizations can be used to create misleading narratives. Using a simple dataset on student sleep patterns, I created both deceptive and ethical charts to show how design choices affect interpretation.

AI-Generated Charts

Notes

The deceptive chart uses techniques such as a truncated y-axis and misleading color encoding to exaggerate a small increase in sleep hours. The ethical version presents the same data with a neutral scale and clearer context.
