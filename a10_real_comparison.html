Assignment 10 — Deceptive vs. Ethical Comparison

Target Grade: B

This page compares two D3 visualizations built from the same real dataset of McDonald’s breakfast item calories. The deceptive version uses design choices that exaggerate patterns and imply a stronger conclusion, while the ethical version presents the same values with clearer context and more neutral framing.

Deceptive Chart

This version reorders the data to imply progression, uses a truncated y-axis to exaggerate differences, and applies a red risk-style color scale that frames higher calorie values as increasingly severe.

Ethical Chart

This version keeps the same dataset but uses original labels, a y-axis starting at zero, and neutral colors so the values can be interpreted with more accuracy and less framing bias.

What Changes Between Them

  • The deceptive chart makes the calorie differences look more dramatic than they are.
  • The deceptive chart also suggests a rising pattern by reordering the items.
  • The ethical chart shows the same data without forcing that interpretation.
  • Both charts use the same values, but the design choices change the message significantly.
