Assignment 10 — Misleading Visualizations

Using a real dataset on McDonald’s breakfast item calories

Target Grade: B

Overview

This version of Assignment 10 uses a real-world dataset based on calorie values from McDonald’s breakfast menu items. The goal is to show how the same dataset can be used to create both misleading and ethical visualizations depending on design choices, framing, and presentation.

Notes

The deceptive visualizations in this version use techniques such as reordering data to imply progression, truncated or compressed scales, dramatic framing, and color choices that suggest risk or severity. The ethical version presents the same calorie data with more accurate context and neutral design.
