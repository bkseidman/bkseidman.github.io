## Project Overview

### Invented Dataset (C Requirements)
- Created a small, realistic dataset (student sleep hours)
- Built a misleading narrative (300+ words)
- Generated 2 AI-based charts
- Recreated a deceptive and ethical version in D3

### Real Dataset (B Requirements)
- Used a real dataset based on McDonald’s breakfast item calorie values
- Filtered to 10 items for clarity and consistency
- Wrote a 400–500 word misleading narrative using a professional tone
- Generated 3 AI-based charts with captions and fabricated sources
- Built deceptive and ethical D3 charts using:
  - truncated axis
  - data reordering
  - misleading color framing
- Included an interactive tooltip
- Wrote a manipulation analysis comparing both versions
